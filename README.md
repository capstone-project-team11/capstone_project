- 전체기한 : 2020. 04 02 ~ 2020. 11. 27
  - 1차 스프린트 : 2020. 04. 02 ~ 2020. 05. 03
  - 2차 스프린트 : 2020. 05. 04 ~ 2020. 07. 02
  - 3차 스프린트 : 2020. 07. 03 ~ 2020. 08. 10
  - 4차 스프린트 : 2020. 09. 01 ~ 2021. 10. 01
